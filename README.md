**For setup**

 - Run `npm install`
 - Copy `exampleConfig.js` to `config.js`
   - Add users api key to `cred`
   - Update flow

**To run**

 - `bash startOrnithology`
   - `Ctrl+c Ctrl+c` to kill

**For es6 compiling**

 - `bash startBabel`
   - command uses bable to compile files then starts the script
